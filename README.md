# Hosted Payment Page - Responsiveness & Mobile UX Audit (UX-2026-0611)

Companion to defect report DR-2026-0611. Documents mobile responsiveness and UX
findings for the Chipcard / Payten / MerchantSafe Unipay hosted payment page,
measured empirically with Playwright on the WebKit (Safari) engine, against
official standards (WCAG 2.2, Apple HIG, Google web.dev).

- **Report:** `index.html` (bilingual EN / SR)
- **Reference implementation:** `demo/` - an accessible, responsive, Payten-branded
  payment page with New card and Saved card flows and official card scheme artwork.

All sources are listed at the bottom of the report.
